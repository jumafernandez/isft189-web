## Descripción

La Tecnicatura Superior en Desarrollo de Software, **nueva para 2026**, forma programadores y programadoras con capacidad de crear soluciones digitales completas. La carrera combina fundamentos sólidos de programación con las tecnologías más demandadas en el mercado laboral actual.

## Perfil del egresado

El egresado estará capacitado para:

- Desarrollar aplicaciones web frontend y backend
- Crear aplicaciones móviles para Android e iOS
- Aplicar metodologías ágiles (Scrum, Kanban) en equipos de desarrollo
- Gestionar control de versiones con Git y GitHub/GitLab
- Diseñar e implementar APIs RESTful y servicios web

## Salidas laborales

- Desarrollador/a web frontend o fullstack
- Programador/a de aplicaciones móviles
- Desarrollador/a backend de APIs y microservicios
- Técnico/a de soporte y mantenimiento de software
- Freelancer en desarrollo de soluciones digitales
